Leetcode 2. 两数相加

Posted on 2020-04-19 In Code , Leetcode Views:

Leetcode第2题：链表按位相加两个非负整数。难度: Medium.

题目

给出两个 非空 的链表用来表示两个非负的整数。其中，它们各自的位数是按照 逆序 的方式存储的，并且它们的每个节点只能存储 一位 数字。

如果，我们将这两个数相加起来，则会返回一个新的链表来表示它们的和。

您可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例：

输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807

来源：Leetcode 2. 两数相加

错误1

提交后出现解答错误：

输入： [2,4,3] [5,6,4]

输出： [7,0,8,0]

预期结果： [7,0,8]

显然是尾部多加了一个节点的问题。解决方法有三个：

  1. 循环结束后删除尾节点。

    这种方法虽然不会增加复杂度，但仍要遍历一遍结果链表，给人感觉没有必要。

  2. 循环中每次添加节点都对l1和l2的状态进行判断，若二者的next节点都为NULL，则不添加新节点。

    每次循环都进行判断，势必降低程序执行速度。

  3. 循环中添加的不是下一个节点，而是当前节点。也就是root节点中不存储计算结果，真正的结果链表从root->next开始。

方法3自然最优。
